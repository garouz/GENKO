# コンポーネント設計書 - GENKO Frontend

**バージョン**: 0.1.0

## 主要コンポーネント

### FileUploader
- Word(.docx)ファイルアップロード
- Drag & Drop 対応
- クリック選択

### ManuscriptList
- 原稿一覧表示
- 選択状事表示
- 処理状況表示

### CheckResults
- 検查結果表示
- 指摘事項リスト
- フィルタリング・ソート

### DictionaryUploader
- CSV/JSON辞書アップロード

### DictionaryList
- 登録済み辞書一覧
- 項目数表示
